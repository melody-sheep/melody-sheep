# 🚀 Featured Project: ChemStock
### **QR-Enabled Mobile Inventory Management System**

**ChemStock** is a mobile-first solution designed to modernize the inventory operations of **Cospachem Products (A and Aimee Laboratories)**. By replacing manual, error-prone paper trails with digital accountability, the system empowers distributed sales teams with real-time tracking and automated loss prevention.

### 💡 The Core Problem
* **Accountability Gaps:** Unverifiable handovers leading to stock shrinkage.
* **Operational Inefficiency:** Manual stock counting consuming hours of labor.
* **Financial Leakage:** Significant annual losses due to tracking inaccuracies.

### 🛠️ The Solution
* **📱 Mobile-First Operation:** Built with **React Native (Expo)** for seamless field deployment.
* **🔍 QR-Based Tracking:** Automated batch identification ensuring FIFO compliance.
* **📍 Geotagged Chain of Custody:** Verifies exact handover locations using GPS.
* **📸 Photo Proof:** Enforced timestamped photo evidence for all handovers.
* **⚠️ Real-Time Alerts:** Immediate variance flagging via **Supabase Real-time**.
